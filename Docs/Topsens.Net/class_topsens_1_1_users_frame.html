Vector2 Topsens.UsersFrame.MapTo2D ( Vector3  position3D)

Obtains a projection of a skeleton joint in a depth camera space on the two-dimensional depth image.

The screen coordinate origin point (0, 0) is located at the upper-left corner of the screen.

Parameters

Vector4 Topsens.UsersFrame.GroundPlane

The plane equation of the ground detected in the scene.

The four elements represent the four coefficients A, B, C and D in the plane equation Ax+By+Cz+D=0.

◆ Projection

Vector4 Topsens.UsersFrame.Projection

Projection parameters.

◆ Skeletons

Skeleton [] Topsens.UsersFrame.Skeletons

The collection of user skeleton data.

Each element corresponds to all identified skeleton joints of a user. The effective length of this array is UserCount.

◆ Timestamp

The data timestamp in milliseconds.
