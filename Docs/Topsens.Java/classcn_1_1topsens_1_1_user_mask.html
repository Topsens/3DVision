Detailed Description

User mask image.

Remarks
Contains mask information of the identified user in the depth image or color image.

Member Data Documentation

◆ Height

int Height

Mask image height in pixels.

It is equal to the corresponding depth image height.

◆ Pixels

short [] Pixels

Mask image pixel collection, array length is Width * Height.

Each pixel values is equal to the user ID to which the corresponding pixel in the depth image belongs. For example, if the mask pixel value of x=100 and y=100 is 1, the depth pixel corresponding to x=100 and y=100 in depth image belongs to the user body with ID=1.

◆ Width

int Width

Mask image width in pixels.
