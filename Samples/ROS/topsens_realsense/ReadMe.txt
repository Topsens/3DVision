How to use Intel RealSense?
0. Plugin RealSense sensor;
1. Install RealSense library: sudo apt install ros-kinetic-realsense2-camera;
2. Install the library of ros-kinetic-topsens_x.x.x-0xenial_amd64.deb;
3. Install the sample of ros-kinetic-topsens-realsense_x.x.x-0xenial_amd64.deb;
4. Start a new terminal, and run "roscore";
5. Start a new terminal, and run "roslaunch topsens_realsense topsens_realsense.launch" to start the sample.
